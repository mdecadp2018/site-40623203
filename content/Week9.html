Week9

Conduct final project grouping and Mid-term Self-assessment results

進行期末專案分組及打期中自評成績

How to conduct final project grouping：

如何進行期末專案分組：

1. A group of up to six people, sitting in the first three and back three seats

至多六人一組，按照前三後三的座位就坐

2. Confirm the class and group corresponding to the seat. The group number is from the front stage. The first area is itting in the first three and back three seats on the right for the first group, the middle group is the second group, and so on.

確認座位對應的班級與組別，組別序號從面對講台，右邊第一區前三後三為第一組，中間為第二組，依此類推

3. The first group code of class B is bg1, the second group is bg2, and so on. The first group code of class A is ag1, the second group is ag2, and so on.

乙班第一組代號為 bg1，第二組為 bg2 依此類推，甲班第一組代號為 ag1，第二組為 ag2 依此類推

4. Log in to Github with the corresponding number of the student number.

以學號對應帳號登入 Github

5. According to the github classroom project warehousing link, add the corresponding groups in order.

根據 github classroom 期末專案倉儲連結，依序加入各對應組別

Second class final project Github：

二乙期末專案倉儲：

First Group、Second Group、Third Group、Fourth Group、Fifth Group、Sixth Group、Seventh Group、Eighth Group

Final Project Independent Study Focus：

期末分組專題重點：

1. Each group selected a multi-part mechanical and electromechanical integration topic

各組選定一個多零件機電資整合專題題目

2. Using collaborative writing CMSimfly, Reveal.js and Pelican Blog to complete project planning, content management, newsletter and blog content for collaborative processes

以協同寫作方式利用 CMSimfly、Reveal.js 與 Pelican Blog 完成專案規劃、內容管理、簡報及網誌內容各組以協同流程

3. Complete component design mechanical component design and analysis on Onshape, use Flask Internet and PyQt5 program to construct auxiliary computing module or system circuit and control program design - use Arduino and Tinkercad to complete multi-part mechanical and electrical integration system simulation - using V-rep to complete with Lua or Remote API Python program (optional) to complete the project with 3D printing components

在 Onshape 上完成零組件設計機械零組件設計與分析運算 - 採用 Flask 網際與 PyQt5 程式建構輔助運算模組或系統電路與控制程式設計 - 採用 Arduino 與 Tinkercad 完成多零件機電整合系統模擬 - 採用 V-rep 結合 Lua 或 Remote API Python 程式完成(選項) 以 3D 列印零組件完成專題實作
