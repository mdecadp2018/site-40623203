LP: College - Week 3 (“Cantilever Clamp” Assembly)

1. What is the difference between a fixed and a moveable assembly in Onshape ( Onshape 中固定組件和可移動組件之間有什麼區別 ) ? 

A Fixed assembly is created using the Top-Down Design method in a Part Studio and is static, while a Moveable assembly is built in an Assembly tab using Mates and can be used to visualize motion ( 固定裝配使用零件中的自頂向下設計方法創建，並且是靜態的，而可移動裝配合使用在裝配選項卡中構建，可用於可視化運動 ) .

2. What is so unique about the Mate Connector in Onshape ( Onshape中的配合連接器有什麼獨特之處 ) ?

It is a single feature which allows for the full definition of all 6 degrees of freedom. It is the bases for how mates are defined ( 它是一個單一的功能，可以完全定義6個自由度。 它是如何定義配合的基礎 ) .

3. How many degrees of freedom do the following Mates have ( 多少自由度執行以下配合 ) :

a. Fastened - 0 ( 固定 - 0 )

b. Revolute - 1 ( 迴轉 - 1 )

c. Slider - 1 ( 滑塊 - 1 )

d. Planar - 3 ( 平面 - 3 )

e. Cylindrical - 2 ( 圓柱形 - 2 )

f. Ball - 3 ( 球 - 3 )

4. When a mate is initially defined, does Onshape solve the entire model, or just that mate ( 當最初定義配合時，Onshape 會整個模型一起定義，還是只有那個配合 ) ?

Just that mate ( 只有那個配合 ) .

5. What is a Relation ( 什麼是關係 ) ? Provide 2 examples. 

It allows degrees of freedom to be defined between two existing mates ( 允許在兩個現有配合之間定義自由度 ) . Examples are Gear, Rack & Pinion, Screw, and Linear ( 例如齒輪和齒輪或齒條，螺釘和線性 ) .
