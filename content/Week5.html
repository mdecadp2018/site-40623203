Week5

Onshape：2-1-Lesson Plan.pdf

I. Introduction of Pyslvs Computer Aided Mechanism Simulation and Synthesis Kit

一、Pyslvs 電腦輔助機構模擬與合成套件介紹

1. Introducing the walking mechanism

介紹行走機構

2. Analysis by Pyslvs

用 Pyslvs 分析

3. You must understand the characteristics before designing

做設計前必須先了解其特性

4. Through simulation, you can understand the position and size of the front and rear feet and whether there is interference.

透過模擬可以了解其前後腳的位子和尺寸以及有沒有干涉

5. After analysis, drawn by Onshape

分析後，用 Onshape 繪製出來

6. When Pyslvs changes any place, the size of the back will change.

在 Pyslvs 改變任何一個地方，後面的尺寸就會跟著變動

二、Pelican blog

1. Repeating slug will cause problems

重複 slug 會產生問題

2. Execute pelican markdown -o blog -s local_publishconf.py under the folder to know where the error is.

將 pelican markdown -o blog -s local_publishconf.py 在資料夾底下執行，才能知道錯在哪

3. Change slug to something different to avoid errors

將 slug 改為不同，避免錯誤

4. Execute pelican markdown -o blog -s local_publishconf.py once under the folder and you will see that the error has been resolved.

再將 pelican markdown -o blog -s local_publishconf.py 在資料夾底下執行一次，會發現錯誤已解決

三、Onshape

 1. To explain the first week and the second week

第一週到第二週講解

2. Use Merriam-Webster to check English words that are not possible

用 Merriam-Webster 查不會的英文單字

3. See the teacher's handout learning operation Onshape

看老師給的講義學習操作 Onshape

4. Sharing experiences with team members in the tenth week can speed up learning

跟第十週所組的隊員一同分享經驗，可以加速學習

5. There are eight topics to choose from：3D printing, walking gear, steel ball system, robotic arm, virtual hand football, mechanical timer, folding bike and formula car

有八個題目可以選擇：3D 列印、行走機構、鋼球運動系統、機械手臂、虛實手足球、機械計時器、摺疊自行車及方程式賽車
